# 모스크바

소규모 교전 맵으로 중앙 거점에서 진행되는 처절한 쟁탈 싸움을 하는 맵입니다.

## 맵 정보

| 속성 | 값 |
|------|-----|
| 크기 | 소형 (192x384) |
| 테마 | 도심 / 비 |
| 지원 모드 | 쟁탈전, 보병전 |
| 최소 인원 | 2명 (1 vs 1) |
| 추천 인원 | 12명 (6 vs 6) |

## 특징

- 맵 중앙 거점을 필두로 대립하는 구조
- 스폰 바로 앞 건물들에서 제공되는 저격 포인트

## 전략 팁

중앙 거점에서 진지를 구축해 건너편 건물들에서 오는 견제를 수비하세요.
