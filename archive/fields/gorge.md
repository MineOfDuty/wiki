# 협곡

중앙 협곡과 눈이 내린 아름다운 배경에서 전투를 경험하세요.

## 맵 정보

| 속성 | 값 |
|------|-----|
| 크기 | 소형 (256x256) |
| 테마 | 산악 / 눈 |
| 지원 모드 | 습격전, 보병전 |
| 최소 인원 | 4명 (2 vs 2) |
| 추천 인원 | 12명 (6 vs 6) |

## 특징

- 초반 거점에서 즐길 수 있는 복잡한 전투
- 중앙 거점에 협곡과 양쪽 지대가 있는 구조
- 후반 거점에서 처절한 수비를 경험

## 전략 팁

공격팀은 중앙 거점 능선을 점령하고 수비팀은 마지막 거점 건물 윗층을 사수하세요.
