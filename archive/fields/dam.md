# 댐

거대한 댐과 험준한 산악 지형을 배경으로 한 대형 전투 맵입니다.

## 맵 정보

| 속성 | 값 |
|------|-----|
| 크기 | 대형 (384x384) |
| 테마 | 산악 / 댐 |
| 지원 모드 | 쟁탈전, 장비전 |
| 최소 인원 | 12명 (6 vs 6) |
| 추천 인원 | 24명 (12 vs 12) |
| 배치 장비 | AH-64, Mi-28, MH-6, 발칸 |

## 특징

- 맵 끝에 위치한 거대한 댐 구조물이 맵의 핵심 랜드마크
- 댐 상부 통로를 통한 고저차 전투 가능
- 각 진영별 헬리콥터 지원과 댐 상부에 있는 방공 환경 제공
- 불리한 팀에게 공수낙하 이벤트 지원

## 전략 팁

댐 상부는 전장 전체를 조망할 수 있는 최고의 저격 포인트입니다. 하지만 노출 위험이 크므로 팀원의 엄호가 필수적입니다. 산악 지역의 건물을 활용한 거점 확보가 승패의 핵심입니다.
