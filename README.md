## 설명

**MineOfDuty**는 마인크래프트 바닐라에서 접속 가능한 FPS 콘텐츠 서버이다.  

수준 높은 애니메이션과 건플레이 및 다양한 장비와 이들이 돌아다닐 수 있는 필드를 제공한다.  

더 구체적인 사항은 위키와 커뮤니티에서 확인하시길 바라겠습니다.
